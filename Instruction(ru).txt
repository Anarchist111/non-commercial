После запуска приложения — оно создаст пустой файл DishList.txt по пути /storage/emulated/0/WWWE/ (Внутренняя память устройства).
В корне ветки main находится уже заполненный файл "DishList.txt". Для интеграции списка в приложение необходимо переместить с заменой этот файл по вышеуказанному пути.
Модификация файла DishList.txt:
    - секции разделены знаком ";". Всё, что идёт после этого знака — игнорируется.
    - количество секций фиксировано.
    - одно название блюда — одна строка. Будут пустые строки — приложение будет считать их блюдами.
    - количество блюд ограничено только возможностью JVM зарезервировать под них память ( а может и нет:) никто это не проверял:))
